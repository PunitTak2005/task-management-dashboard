# Task Management Dashboard

This is a **Task Management Dashboard** built with **Next.js**, **TypeScript**, **Tailwind CSS**, **PostgreSQL**, and **Redux**. It allows users to manage their tasks efficiently with CRUD operations, including listing, adding, updating status (To Do, In Progress, Done), and deleting tasks.

### Live Application
[Task Management Dashboard](https://task-management-dashboard-zeta-brown.vercel.app/)

---

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).
